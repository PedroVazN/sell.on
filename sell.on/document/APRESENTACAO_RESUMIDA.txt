🚀 SELL.ON - SISTEMA DE GESTÃO COMERCIAL
============================================

SLIDE 1: VISÃO GERAL
====================
• Sistema completo de gestão comercial e vendas
• Plataforma web moderna e responsiva
• Tecnologias: React + Node.js + MongoDB
• Hospedagem: Vercel (Frontend + Backend)
• Segurança: JWT + Validação + Rate Limiting

SLIDE 2: FUNCIONALIDADES PRINCIPAIS
===================================
• Criação de propostas personalizadas
• Gestão de clientes e produtos
• Listas de preços flexíveis (À Vista, Boleto, Cartão)
• Dashboard com métricas em tempo real
• Sistema de notificações e avisos
• Controle de usuários (Admin/Vendedor)

SLIDE 3: SEGURANÇA E CONFORMIDADE LGPD
======================================
• Autenticação JWT obrigatória
• Senhas hashadas com bcrypt
• Validação robusta de dados
• Rate limiting em todas as rotas
• Headers de segurança completos
• Monitoramento de ataques em tempo real
• Logs sanitizados sem dados sensíveis
• CONFORMIDADE LGPD COMPLETA:
  - Dados pessoais criptografados
  - Acesso restrito por roles
  - Auditoria completa de acessos
  - Backup seguro no MongoDB Atlas
  - Exclusão de dados quando solicitado
  - Transparência no uso dos dados

SLIDE 8: CONFORMIDADE LGPD - TRANQUILIDADE TOTAL
================================================
• DADOS PESSOAIS PROTEGIDOS:
  - Criptografia de ponta a ponta
  - Acesso apenas com autenticação
  - Senhas nunca armazenadas em texto
  - Logs sem exposição de dados sensíveis

• CONTROLE DE ACESSO:
  - Usuários só veem seus dados
  - Admin tem acesso controlado
  - Auditoria de todas as ações
  - Sessões com expiração automática

• BACKUP E SEGURANÇA:
  - MongoDB Atlas (servidor seguro)
  - Backup automático diário
  - Criptografia em repouso
  - Conformidade internacional

• DIREITOS DO USUÁRIO:
  - Exclusão de dados quando solicitado
  - Portabilidade dos dados
  - Transparência no uso
  - Consentimento explícito

• IMPLEMENTAÇÕES TÉCNICAS:
  - Sanitização automática de logs

  - Rate limiting contra ataques
